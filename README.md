# Ashutosh Amale - AI & Software Engineering Portfolio

Welcome to the open-source repository for my personal portfolio! I am Ashutosh Amale, a Computer Engineering student specializing in Artificial Intelligence, Data Science, and Full-Stack Software Engineering.

This repository contains the source code for my interactive portfolio, designed to showcase my projects, skills, and experience with a highly dynamic, premium user experience.

## ✨ Features

- **Interactive 3D Hero Section**: A responsive, cinematic hero section with 3D tilting portrait effects, smooth typography reveals, and floating tech elements built using Framer Motion.
- **Custom Particles Background & Cursor**: A lightweight, canvas-based particle engine and custom dynamic cursor for an immersive feel.
- **Embedded AI Maze Simulation**: A fully playable, React-based Autonomous Maze Solving Robot simulation (Project 03) featuring pathfinding algorithms like the Left-Hand Rule and BFS, complete with sensors, telemetry panels, and custom wall drawing.
- **Advanced 3D Project Deck**: An interactive, gesture-controlled swipeable deck to showcase highlighted projects (e.g., Heart Attack Prediction, RetailPro POS).
- **Infinite Marquee & Glassmorphism Design**: Seamless scrolling typography combined with modern UI trends using frosted glass styling (glassmorphism) and a sleek dark neon aesthetic.
- **Responsive Layout**: Designed to look stunning on both desktop and mobile devices.

## 🛠️ Technology Stack

- **Core**: React 18, TypeScript, Vite
- **Styling**: Vanilla CSS (Tailwind CSS removed for customized control), Lucide React & React Icons for iconography
- **Animation**: Framer Motion for scroll-linked animations, spring physics, and complex gesture interactions
- **Deployment**: Configured for standard hosting platforms (Vercel, Netlify)

## 🚀 Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/App.tsx`: The main router and layout wrapper.
- `src/pages/home/Index.tsx`: The primary landing page containing the Hero, Projects, Services, Experience, Skills, and Contact sections.
- `src/components/MazeSolver.tsx`: The complex interactive micro-mouse simulation component.
- `src/index.css`: The central stylesheet containing theme tokens, utility classes, and custom animations.
- `public/`: Static assets including videos, fonts, and images.
